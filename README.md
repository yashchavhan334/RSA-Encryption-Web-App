# 🔐 RSA Generator

A modern RSA Encryption & Decryption Web Application built using **Node.js**, **Express.js**, **HTML**, **CSS**, and **JavaScript**.

This project allows users to generate RSA key pairs, encrypt plaintext using a public key, and decrypt ciphertext using the matching private key.

---

## 🚀 Features

- 🔑 Generate RSA Key Pair
- 🔒 Encrypt Plain Text
- 🔓 Decrypt Cipher Text
- 📋 Copy Public Key
- 📋 Copy Private Key
- 📋 Copy Encrypted Text
- 💾 Download Public & Private Keys
- 🌙 Dark / ☀️ Light Theme
- 💾 Theme Memory (Local Storage)
- 🔢 Character Counter
- ⚠️ Message Length Validation
- 🔔 Toast Notifications
- 🧹 Clear All Fields
- 📱 Responsive Design

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/rsa-generator.git
```

Go into the project

```bash
cd rsa-generator
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```

Open your browser

```
http://localhost:3000
```

---

## 🔒 RSA Workflow

1. Generate RSA Keys
2. Copy the Public Key
3. Enter Plain Text
4. Encrypt using the Public Key
5. Copy the Encrypted Text
6. Paste the Encrypted Text
7. Enter the Matching Private Key
8. Decrypt to retrieve the original message
